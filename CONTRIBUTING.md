# Contributing to QoreDB

Thank you for your interest! As a freelancer-led project, community help is vital.

## Prerequisites
- **Rust**: Latest stable version.
- **Node.js**: v18+ (pnpm recommended).
- **Docker**: Used to spin up local test databases (PostgreSQL, MySQL, MongoDB).

## Local Setup
1. Clone the repo.
2. Install dependencies: `pnpm install`.
3. Start the dev environment: `pnpm tauri dev`.

## Pull Request Process
- Create a feature branch.
- Ensure your code follows the project's Linting rules.
- Run tests: `cargo test` and `pnpm test`.
