# Security Policy

## Supported Versions

Only the latest version of QoreDB is supported with security updates.

## Reporting a Vulnerability

**Please do not open a public issue for security vulnerabilities.**

If you discover a security-related bug, please send an email to [qoredb@gmail.com](mailto:qoredb@gmail.com).
We acknowledge all submissions within 48 hours and will work with you to fix the issue before public disclosure.
