## Description longue (affichée sur la page produit)

### The problem

Database tools haven't kept up with modern developer workflows. DBeaver is slow. pgAdmin feels stuck in 2010. TablePlus is nice but limited. You're switching between 3 tools for 3 databases, and none of them feel *good*.

### What is QoreDB?

QoreDB is a desktop database client that does for databases what VS Code did for editors — a fast, clean, extensible tool that gets out of your way.

Built from the ground up with **Rust** (Tauri 2) and **React 19**, it supports **9 databases** in a single, unified interface:

🔷 **SQL**: PostgreSQL, MySQL, MariaDB, SQLite, SQL Server, CockroachDB, DuckDB
🟠 **NoSQL**: MongoDB, Redis

### Why developers love it

- **Blazing fast** — Rust backend, virtualized data grid, streaming queries. No Electron tax.
- **Local-first** — Your credentials are encrypted in a vault (Argon2). Your data never touches a cloud. Ever.
- **Modern SQL editor** — CodeMirror-based with autocomplete, formatting, snippets, multi-statement execution.
- **Inline editing** — Click any cell, edit, save. Works across SQL and NoSQL.
- **SSH tunneling** — Native support for bastion hosts, key-based auth, jump proxies.
- **Production safety** — Environment classification (dev/staging/prod), read-only mode, destructive query blocking.
- **Full-text search** — Search any string across all tables in your database.
- **ER diagrams** — Interactive entity-relationship visualization.
- **Query library** — Save, organize, tag, and share your queries.
- **Export anything** — CSV, JSON, SQL, HTML, XLSX, Parquet.
- **Dark mode** — Obviously.

### Pro features (optional)

- 🤖 **AI Assistant** (BYOK) — Generate queries, explain results, fix errors with your own API key
- 🔀 **Cross-database federation** — JOIN across PostgreSQL + MySQL in a single query via DuckDB
- 📊 **Visual Data Diff** — Compare datasets side-by-side (prod vs staging, before vs after)
- 🏗️ **Sandbox mode** — Test schema changes locally before generating migrations
- 📈 **Query profiling** — Track slow queries, analyze performance patterns

### Open source

QoreDB's core is Apache 2.0. Free forever. Pro features use BSL-1.1 with a planned conversion to open source.

We believe the best developer tools are built in the open.

## Premier commentaire du Maker

Hey Product Hunt 👋

I'm Raphaël, the solo developer behind QoreDB.

I've been building this for months because I was tired of the same frustration every developer knows: opening DBeaver and waiting... and waiting. Switching to pgAdmin for Postgres, then to Compass for MongoDB, then to RedisInsight for Redis. None of them felt like they were made for how we work today.

→ **The core is open source** (Apache 2.0). Pro features exist for power users and teams, but the free version is a complete, production-ready tool.

I'm a solo founder bootstrapping this, so every upvote and piece of feedback means a lot. Try it, break it, tell me what sucks — I read everything.

What database tool do you currently use, and what drives you crazy about it? I'd love to hear.
